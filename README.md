# osu-buckeye-marketplace
An e-commerce platform design project for Ohio State University focused on user personas, journey mapping, feature development, and user stories.


AI Tool Usage
For this milestone I used AI tools like ChatGPT and GitHub Copilot to help me while building the project. I did not just copy code. I used the tools mostly to help explain things and fix problems when I got stuck.
What I asked AI to help with
I used AI for a few parts of the project.
Help creating the ProductsController in the .NET API
Generating sample product data for the in memory product list
Helping me figure out how to show the products in the React product list page
Fixing problems with image sizes so the cards looked the same
Checking if my API endpoints matched the assignment requirements
Example prompts I used
Here are some examples of questions I asked.
"How do I make a .NET API endpoint that returns a list of products"
"Generate sample marketplace products with id title description price category seller name posted date and image url"
"How can I make all images the same size in React cards"
"Does my ProductsController meet the requirements for this assignment"
What I accepted modified or rejected
Accepted
I used the basic structure for the ProductsController and some of the example product objects that were generated.
Modified
I changed the product data and added more items so there were at least 8 products and multiple categories. I also changed image links and sizes so everything looked consistent in the UI.
Rejected
Some layout suggestions did not match my project so I did not use them. Some generated product data was also missing fields so I fixed it myself.
Where I used my own judgment
Even though AI helped me, I still checked everything before using it. I made sure the API returned the correct status codes and that the project followed the rubric. I also tested the React frontend to make sure it was actually pulling data from the API and not using hardcoded data.
AI mostly helped me understand things faster and fix issues while I was working on the project.
